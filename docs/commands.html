Common Terminal Commands

Command Result Example
ssh (user@domain.name) Connect via secure shell $ ssh eyork@142.93.199.169
sftp (user@domain.name) Connect via SFTP $ sftp eyork@142.93.199.169
sudo (command) Do as super-user (root) $ sudo mkdir foo
cd (directory name) Change directory $ cd foo
. Current directory $ cp foo.txt .
.. Directory one level up $ cd ..
ls List files and directories $ ls
ls -al Lists all in long form $ ls -al
pwd Print working directory $ pwd
mkdir (directory name) Make a directory $ mkdir foo
rmdir Remove a directory $ rmdir foo
touch (filename) Create a file $ touch foo.txt
rm (filename) Remove (delete) a file $ rm foo.txt
mv (file1) (file 2) Move (rename) a file $ mv foo.text ../foo.txt
cp (old) (new) Copy a file $ cp foo.txt ..
nano (filename) Edits a file using nano $ nano foo.txt
chmod (perms) (file) Change the mode (perms) $ chmod 755 foo.txt
chown Change the owner $ chown root:root foo
